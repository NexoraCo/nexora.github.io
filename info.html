About Nexora

How It Was Made

Nexora was designed to be a fast, clean launcher for indie gamers. No telemetry, no bloat, and fully modular.

  • Custom WPF UI for smooth, modern visuals
  • MVVM architecture separating UI, logic, and data
  • JSON-based game database for easy management
  • GitHub API integration for future auto-updates
  • NSIS installer with user-level and portable modes

Tech Stack

  • .NET 8 (self-contained runtime)
  • WPF for UI and dynamic layouts
  • MVVM Architecture (Models, ViewModels, Services)
  • GitHub Releases API for version tracking
  • JSON-based database (games.json)

Lore

Nexora isn’t just a launcher — it’s a minimalist hub for indie gaming. Its philosophy:

  • Born from the frustration of slow, bloated launchers
  • Mission: give gamers full control over their library
  • Design: minimalism, speed, and clarity
  • Vision: modular ecosystem, optional cloud sync, plugins, and auto-updates
